ğŸ“¦ Installation

  1. Houdini Packages Setup
  2. Replace DIR_PATH key-value path/to/source/folder with current path of this repo in explorer inside HFilePathManager_tool.json file.
  3. Copy the HFilePathManager_tool.json package file inside the same houdini's packages/ directory.:
    • On Windows:
      C:/Users/<YourName>/Documents/houdiniXX.X/packages/
    • On Linux: /home/<YourName>/houdiniXX.X/packages/
  4. Launch the Tool
  5. Launch Houdini
  6. Go to Windows â†’ Python Panel
  7. Click on File Path Manager
  8. If all goes well it will show you the panel.
